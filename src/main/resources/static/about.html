说明

目前该应用处于开发测试阶段。该应用纯属于个人兴趣爱好，至于做到什么程度？未知！

目前所实现的产品功能有：

  • 1.登录注册
  • 2.文章页面查看
  • 3.文章编写与上传（MarkDown）
  • 4.用户个人中心
  • 5.文章关键字搜索/题目搜索
  • 6.热搜排行榜（一天换一次）
  • 7.最新推荐(半小时更新一次)
  • 8.删除自己的文章
  • 9.查看用户空间
  • 10.文章点赞功能
  • 11.日志记录(开发)

目前尚未实现的功能（这里只列出想到的功能）：

  • 1.评论功能
  • 2.文章修改功能
  • 3.支持更多MarkDown语法（目前尚不支持上传图片、表格、数学公式,可能还有尚未发现）
  • ...
